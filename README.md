# ESP_RELAY
Mạch Phát Triển ESP32-WROOM Điều Khiển 4 Relay Wifi BLE hỗ trợ cổng I/O và giao tiếp UART, có thể sử dụng IDE Arduino,đèn báo nguồn trên mạch, đèn led lập trình và đèn báo relay.
